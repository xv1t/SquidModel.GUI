Option Name:wccp2_address
Replaces:
Requires:--enable-wccpv2
Default Value:wccp2_address 0.0.0.0
Suggested Config: 


	Use this option if you require WCCP to use a specific
	interface address.

	The default behavior is to not bind to any specific address.
